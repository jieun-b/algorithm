## 핵심 개념
- **조합론**: 순열, 조합, 중복 순열, 중복 조합
- **수열**: 등차수열, 등비수열, 피보나치 수열
- **진법**: 2진법, 8진법, 16진법 변환
- **나머지 연산**: 모듈러 연산, 중국인의 나머지 정리
- **비트 연산**: AND, OR, XOR, 시프트 연산
- **게임 이론**: 승리 전략, 필승법
