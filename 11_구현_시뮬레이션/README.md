- **구현**: 문제의 요구사항을 정확히 코드로 옮기는 것
- **시뮬레이션**: 실제 상황을 컴퓨터로 모델링
- **문자열 처리**: 문자열 조작, 패턴 매칭
- **수학적 사고**: 공식 도출, 규칙 찾기
- **자료구조 활용**: 적절한 자료구조 선택과 활용
- **알고리즘 조합**: 여러 알고리즘을 조합하여 해결
